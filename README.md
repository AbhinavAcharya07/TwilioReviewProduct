# TwilioReviewProduct
Steps
1.Go to the website https://twilio-review-product.vercel.app.
2.Scan the scanner and send the default message by the twilio to initiate the conversation.
3.Then answer the every query.
4.Now your review is saved in database and reflected in the frontend once you refresh the page


(note:this scanner has daily message limit of 50 only)
